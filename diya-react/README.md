DIYA's frontend follows a **State-Driven Animation** architecture:

1.  **Component-Based UI**: Atomic design principles applied to React components.
2.  **Animation Controllers**: GSAP timelines are integrated into `useEffect` and triggered by React state changes (e.g., transition from `progress` to `success` in OAuth).
3.  **Dynamic Theming**: Brand-specific colors and backgrounds are injected via CSS variables and React hooks.
4.  **Spotlight Isolation**: A central "Spotlight" modal system that uses backdrop filters and perspective transforms to focus user attention.
